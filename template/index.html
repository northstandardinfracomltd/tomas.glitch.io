Project pour les professionnels

Project est une entreprise spécialisée dans le BTP, nous sommes en capacité de travailler avec les professionnels pour les accomapgner à réaliser leurs projets. Nous proposons une offre spécifique comprenant notamment le chiffrage TCE, la réponse aux appels d'offres, le suivi de chantier, la création de plans 2D et 3D, les dossiers techniques, les dossiers administratifs DOE, ainsi que la prise en charge Qualibat.



Le tarif est indicatif et débute à partir de 450€ HT. N'hésitez pas à contacter notre équipe pour planifier une première réunion afin de faire le point sur vos besoins et votre projet. Nous serons ravis de vous accompagner dans la réalisation de votre projet.
